The folllowing text files contain the following 

1.BASEBOARD.txt -  Base board (motherboard or system board) management

2. BIOS.txt -  Basic input/output services (BIOS) management.

3.BOOTCONFIG.txt - Boot configuration management.

4. CDROM.txt - CD-ROM management.

5. COMPUTERSYSTEM.txt -  Computer system management.

6. CSPRODUCT.txt -  Computer system product information from SMBIOS.

7. DATAFILE.txt - DataFile Management.

8. DCOMAPP.txt -  DCOM Application management.

9. DESKTOP.txt - User's Desktop management.

10.  DESKTOPMONITOR.txt - Desktop Monitor management.

11. DEVICEMEMORYADDRESS.txt - Device memory addresses management.

12. DISKQUOTA.txt - Disk space usage for NTFS volumes. 

13 . DMACHANNEL.txt -  Direct memory access (DMA) channel management.

13. ENVIRONMENT.txt - System environment settings

14. FSDIR.txt - Filesystem directory entry management.

15. GROUP.txt - Group account management.

16. IDECONTROLLER.txt - IDE Controller management.

17. IRQ.txt -  Interrupt request line (IRQ) management.

18. JOB.txt - Provides  access to the jobs scheduled using the schedule service.

19. LOADORDER.txt -  Management of system services that define execution dependencies.

20. LOGICALDISK -  Local storage device management.

21. MEMCACHE.txt - Cache memory management.

22. MEMORYCHIP.txt - Memory chip information.

23.MEMPHYSICAL .txt  -  Computer system's physical memory management.

24. NETCLIENT.txt - Network Client management.

25. NETLOGIN.txt -  Network login information (of a particular user) management.

26. NETPROTOCOL.txt - Protocols (and their network characteristics) management.

27. NETUSE.txt - Active network connection management.

28. NIC.txt - Network Interface Controller (NIC) management.

29.NICCONFIG.txt - Network adapter management.

30. NTDOMAIN.txt - NT Domain management.

31. NTEVENT.txt - Entries in the NT Event Log.

32. NTEVENTLOG.txt - NT eventlog file management.

33.ONBOARDDEVICE .txt -  Management of common adapter devices built into the motherboard (system board).

34. OS - Installed Operating System/s management.

35. PAGEFILE.txt - Virtual memory file swapping management.

36. PAGEFILESET.txt - Page file settings management.

37. PARTITION.txt -  Management of partitioned areas of a physical disk.

38. PORT.txt - I/O port management.

40. PORTCONNECTOR.txt - 
Physical connection ports management.

41. PRINTER - Printer device management.

42.PRINTERCONFIG.txt -  Printer device configuration management.

43. PRINTJOB.txt - Print job management.

44.PROCESS.txt - Process management.

45. PRODUCT.txt - Installation package task management.

46.QFE.txt - Quick Fix Engineering.

47.  QUOTASETTING .txt - Setting information for disk quotas on a volume.

48. RDACCOUNT.txt - Remote Desktop connection permission management.

49. RDNIC.txt - Remote Desktop connection management on a specific network adapter.

50. RDPERMISSIONS .txt - Permissions to a specific Remote Desktop connection.

51. RDTOGGLE.txt - Turning Remote Desktop listener on or off remotely.

52. RECOVEROS .txt - Information that will be gathered from memory when the operating system fails.

53. REGISTRY.txt - Computer system registry management.

54. SCSICONTROLLER.txt - SCSI Controller management.

55.  SERVER.txt
- Server information management.

56. SERVICE.txt -     Service application management.

57.SHADOWCOPY.txt - Shadow copy management.

58. SHADOWSTORAGE.txt - Shadow copy storage area management.

59.  SHARE.txt - Shared resource management.

60. SOFTWAREELEMENT .txt - Management of the  elements of a software product installed on a system.

61. SOFTWAREFEATURE .txt -  Management of software product subsets of SoftwareElement.

62. SOUNDDEV.txt - Sound Device management.
 
63.STARTUP.txt - Management of commands that run automatically when users log onto the computer system.

64. SYSACCOUNT.txt - System account management.

65. SYSDRIVER.txt - Management of the system driver for a base service.

66. SYSTEMENCLOSURE.txt - Physical system enclosure management.

67. SYSTEMSLOT.txt - Management of physical connection points including ports,  slots and peripherals, and proprietary connections points.

68. TAPEDRIVE.txt - Tape drive management.

69. TEMPERATURE.txt - Data management of a temperature sensor (electronic thermometer).

70. TIMEZONE.txt - Time zone data management.

71. UPS.txt - Uninterruptible power supply (UPS) management.

72. USERACCOUNT.txt - User account management.

73. VOLUME.txt - Local storage volume management.

74. VOLTAGE.txt - Voltage sensor (electronic voltmeter) data management.

75. VOLUMEQUOTASETTING.txt - Associates the disk quota setting with a specific disk volume.

76. VOLUMEUSERQUOTA.txt - Per user storage volume quota management.

77. WMISET.txt - WMI service operational parameters management.
